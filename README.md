
# Prueba Técnica de Frontend en Laravel con Tailwind CSS

Esta es una guía paso a paso para instalar y ejecutar la prueba técnica de frontend en un proyecto Laravel utilizando Tailwind CSS para el estilo.

## Requisitos previos

- PHP instalado en tu sistema.
- Composer instalado en tu sistema.
- Node.js y npm instalados en tu sistema.

   


## Repositorio

Clona este repositorio en tu máquina local utilizando el siguiente comando:

```bash
  git clone https://github.com/codewithsebas/testTechnicalFront.git
```
    
## Instalación

```bash
  composer install
```

luego 

```bash
  npm install
```

## Ejecucion

```bash
  npm run dev
```

luego 

```bash
  php artisan serve
```


## Visitar 

```bash
  http://localhost:8000
```
